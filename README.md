## 🎯 Purpose

The goal of this project is to provide:

- Clear, structured guidance for ransomware incident response  
- Practical scripts to support containment, triage, and evidence collection  
- Repeatable workflows for SOC and security engineering teams  
- Open-source materials for learning, training, and real-world use
